# CBC-Transition-Readiness-Analysis-Kenya
Education analytics project evaluating CBC implementation readiness across Kenya through infrastructure, staffing, and resource distribution analysis.

## Objectives

- Assess infrastructure readiness for CBC implementation
- Evaluate teacher availability and capacity
- Analyze learner-to-teacher ratios across regions
- Identify county-level disparities in preparedness
- Develop a readiness assessment framework
- Provide evidence-based recommendations for education stakeholders

---

## Tools & Technologies

- **Google Sheets** – Data cleaning and exploratory analysis
- **Power BI** – Interactive dashboards and visualizations
- **Python (Google Colab)** – Statistical analysis and data processing

---

## Methodology

### Data Preparation
- Data collection and validation

### Exploratory Analysis
- Descriptive statistics

### Dashboard Development
- Interactive Power BI visualizations

### Statistical Analysis
- Correlation analysis

---

## Key Findings

- Significant differences exist in CBC readiness across counties.
- Infrastructure availability remains a major challenge in several regions.
- Teacher shortages continue to affect implementation capacity.
- Learner-to-teacher ratios vary considerably across counties.
- Resource allocation gaps contribute to unequal readiness levels.

---

## Recommendations

1. Prioritize infrastructure investments in low-readiness counties.
2. Increase teacher recruitment and deployment in underserved regions.
3. Improve distribution of CBC learning resources.
4. Establish continuous readiness monitoring systems.
5. Use data-driven planning to support equitable implementation.
